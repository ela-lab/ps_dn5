# Redovalnica Paket
Paket redovalnica predstavlja preprosto simulacijo šolskega ocenjevanja, ki omogoča dodajanje ocen, izpis redovalnice in računanje povprečja. Aplikacija uporablja paket urfave/cli, ki omogoča konfiguracijo programa preko ukazne vrstice.

Aplikacija omogoča, da uporabnik s pomočjo treh stikal določi pravila ocenjevanja:
- stOcen – minimalno število ocen, potrebnih za pozitivno oceno
- minOcena – najmanjša dovoljena ocena
- maxOcena – največja dovoljena ocena
